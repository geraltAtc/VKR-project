# lite.travel

`lite.travel` — прогрессивное веб-приложение (PWA) в формате электронного гида для туриста.

## О проекте

Приложение помогает сопровождать туриста в поездке через единый интерфейс: вместо разрозненных данных в чатах и файлах пользователь получает структурированную информацию по своему туру.

Система разделена на два контура:
- **Турист**: доступ к своему туру по персональной ссылке, просмотр программы поездки, карты, погодных данных и чек-листа.
- **Администратор**: вход по токену, создание и редактирование туров, достопримечательностей, чек-листов и персональных ссылок.

## Ключевые возможности

- персональные ссылки доступа к туру;
- админ-панель с CRUD-операциями;
- карта на базе **MapLibre** (точки интереса и маршруты);
- интеграция с **OpenWeather API** (с fallback-режимом);
- офлайн-доступ к ранее загруженным данным за счёт PWA-механизмов;
- адаптивный интерфейс и поддержка светлой/тёмной темы.

## Технологии

- **Next.js + React + TypeScript**
- **Supabase (PostgreSQL)**
- **MapLibre + OpenStreetMap**
- **OpenWeather API**
- **Service Worker / PWA**
